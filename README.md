# Installed Capacity Statewise Analysis
This analysis shows about installed energy capacity in various states in India.

📌 Project Overview

This project focuses on analyzing State-wise Installed Power Capacity in India using Python-based data analysis and visualization techniques. The analysis explores the contribution of different energy sources such as coal, gas, hydro, nuclear, diesel, lignite, and renewable energy across Indian states.

The project aims to identify:

Energy distribution patterns
Renewable energy growth
Fossil fuel dependency
State-wise power generation trends
Relationships between different energy sources
📂 Dataset Information

The dataset contains installed power capacity data for Indian states with features including:

State Name
Coal Capacity
Gas Capacity
Hydro Capacity
Nuclear Capacity
Diesel Capacity
Lignite Capacity
Renewable Energy Capacity
Sector Information
Date

🔍 Key Insights
Coal is the dominant energy source in India.
Renewable energy capacity is increasing steadily.
Some states have a balanced clean and fossil fuel energy mix.
Renewable and hydro energy show positive correlation trends.
Industrial states contribute the highest installed capacities.
